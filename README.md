# NOCTRA - Premium Discord Store Bot

A modular, clean-architecture, enterprise-grade Discord Store Bot. 

## Features
- Dynamic Categorization & Variant Management
- Advanced Custom Input Field Builders per product
- Robust Order, Payment, and Ticket Tracking Systems
- Strict Verified-Purchase Review & Rating Engine
- Clean Aesthetic (Dark Purple & Blue Violet Theme, Emoji-free design)

## Deployment on Railway
1. Push this repository to GitHub.
2. Link the repository to Railway.
3. Add environmental variables from `.env.example`.
4. Run!
